Match<TLeft, TRight>(Either<TLeft, TRight>, Action<TLeft>, Action<TRight>)

Run an action based on the value of either.

public static void Match<TLeft, TRight>(Either<TLeft, TRight> either, Action<TLeft> left, Action<TRight> right)

Parameters

either Either<TLeft, TRight>

Result object.

left Action<TLeft>

Function to run when either is Left<TLeft, TRight>.

right Action<TRight>

Function to run when either is Right<TLeft, TRight>.

Type Parameters

TLeft

Left (error / invalid) value type.

TRight

Right (correct / valid) value type.

Remarks

It would be possible to achieve the same thing using switch statements, but doing that would add the potential for not handling both cases (Right<TLeft, TRight> and Left<TLeft, TRight>).

This way there has to be a function to handle both cases.

Exceptions

InvalidEitherTypeException
NullEitherException
